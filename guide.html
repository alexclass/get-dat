get-dat is a tutorial that teaches you how to work with datasets using dat.

Dat is still very much in the early stages of development, but we hope to enable some really cool collaboration workflows.

In this panel you will find step by step instructions for this tutorial. There are navigation links at the bottom of each page.

There is a terminal, file browser, and text editor that is connected to a docker container -- that is a computer, in the cloud but accessible right here in your browser!

When you leave or close this webpage your server will be shut down.

Ok, ready to dive in? Let's go! »
